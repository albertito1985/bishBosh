Bish-Bosh 2.0

Bish-Bosh is a simple number game often used to teach loops and conditionals in programming. The idea is to count from 1 up to a chosen number. For each number:

  • it's divisible by a chosen "Bish" number, the system will generate "Bish".
  • If it's divisible by a "Bosh" number, the system will generate "Bosh".
  • If it's divisible by both, the system will generate "Bish-Bosh".
  • Otherwise, it will display the number itself.

This version of Bish-Bosh was created as part of an assignment for the Lexicon C#.NET course.
